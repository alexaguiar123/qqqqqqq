Encontre informações sobre como usar o sistema e solucionar dúvidas.

Como Funciona?

O Pineapple EcoSense monitora gases em tempo real, utilizando sensores conectados via **cabo USB**. Com a plataforma, você pode visualizar os dados do monitoramento diretamente no painel de controle.

Passos Iniciais

  • Conecte os sensores ao computador usando o cabo USB.
  • Faça login com seu nome e e-mail.
  • Acesse o painel de monitoramento para visualizar os dados.

Como Configurar

Conectando os Sensores

Para iniciar o monitoramento, basta conectar os sensores ao computador via cabo USB. O sistema detectará automaticamente os dispositivos conectados.

Ir para o Monitoramento

Perguntas Frequentes

Como posso visualizar os dados do sensor?

Os dados dos sensores são exibidos em tempo real no painel de monitoramento da plataforma.

O que fazer se o sensor não for detectado?

Verifique se o cabo USB está corretamente conectado e se os drivers do sistema estão atualizados.
